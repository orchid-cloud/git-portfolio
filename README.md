# QA Portfolio

## Overview

This repository contains a collection of QA artifacts demonstrating my skills.

The portfolio includes real-world examples of:

- Bug reporting
- Test case design
- Test planning and strategy
- Exploratory testing
- Basic automation using Cypress

## Repository Structure

### `cypress/e2e`

Contains automated tests written in Cypress.

- `registration.cy.js` — example end-to-end test for user registration

#### How to run tests

```bash
npx cypress open
```

### `web-e-commerce`

#### `test-cases`

Test cases covering core functionality of a web application.

#### `testing-strategy.md`

Web testing strategy.

### `mobile`

#### `bug reports`

Mobile bug reports.

#### `test-plan.md`

Mobile test plan.

#### `exploratory-testing-charter.md`

Session-based exploratory testing example.

#### `assets`

Contains supporting files (screenshots).
