# QA Portfolio

## Overview

This repository contains a collection of QA artifacts demonstrating my skills.

The portfolio includes real-world examples of:

- Bug reporting
- Test case design
- Test planning and strategy
- Exploratory testing
- Basic automation using Cypress and Playwright

## Repository Structure

### `cypress/e2e`

Contains automated tests written in Cypress and JavaScript.

- `registration.cy.js` — example end-to-end test for user registration

### `playwright`

Contains sample automated tests written in Playwright and TypeScript.

Examples include:

- Smoke tests
- Search and filtering validation
- UI element verification
- Responsive/mobile testing

### `web-e-commerce`

#### `test-cases`

Test cases covering core functionality of a web application.

#### `testing-strategy.md`

Web testing strategy.

### `mobile`

#### `bug reports`

Mobile bug reports.

#### `test-plan.md`

Mobile test plan.

#### `exploratory-testing-charter.md`

Session-based exploratory testing example.

#### `assets`

Contains supporting files (screenshots).

## Skills Demonstrated

- Test case design & documentation
- Bug reporting with clear reproduction steps
- Mobile testing (network, gestures, permissions)
- Accessibility testing (keyboard navigation)
- Exploratory testing
- Test automation using Cypress (JavaScript) and Playwright (TypeScript)
- QA documentation (Test Plan, Strategy)
